# 🖨️ Cotizador de Imprenta

Aplicación web para calcular el costo de impresiones según tipo de hoja, tamaño, tipo de impresión y urgencia. El usuario llena un formulario y obtiene el desglose del costo al instante.


## 🛠️ Tecnologías usadas

- HTML, CSS y JavaScript vanilla (frontend)
- Node.js + Express (backend)
- Supabase (base de datos en la nube)


## ▶️ Cómo correrlo

1. Clona el repositorio 
```
https://github.com/owxr11/Cotizador_Imprenta
```
2. Entra a la carpeta `backend`, crea un archivo `.env` con tus credenciales de Supabase:
   -SUPABASE_URL=tu_url
   -SUPABASE_KEY=tu_key
3. Instala dependencias y enciende el servidor:
```bash
   cd backend
   npm install
   npm run dev
```
4. Abre `frontend/index.html` en el navegador (o con Live Server en VS Code)
